# Personal Finance Manager

Backend-приложение для управления личными финансами на Java.

## Функциональность

### Основные функции
- Регистрация и авторизация пользователей
- Учет доходов и расходов  
- Категории транзакций
- Бюджеты по категориям
- Оповещения о превышении бюджета
- Переводы между пользователями
- Сохранение данных в файлы

### Расширенные функции
- Фильтрация транзакций по категориям и периодам
- Экспорт/импорт данных (CSV/JSON)
- Расширенные оповещения (50%, 80%, 90% лимита)
- Редактирование бюджетов и категорий
- Форматированные таблицы и справка

## Архитектура
personal-finance-manager/
├── src/com/finance/
│ ├── model/ # Domain модели
│ ├── service/ # Бизнес-логика
│ ├── repository/ # Интерфейсы хранилищ
│ ├── storage/ # Реализации хранилищ
├── src/test/java/com/finance # тесты
├── data/ # Файлы данных пользователей

## Основные команды
1. Добавить доход
2. Добавить расход  
3. Установить бюджет
4. Редактировать бюджет
5. Показать статистику
6. Показать баланс
7. Фильтр транзакций
8. Перевод другому пользователю
9. Проверить оповещения
10. Экспорт данных
11. Импорт данных
12. Справка
13. Выход

## Пример использования
Добавление дохода: 50000 зарплата "Зарплата за октябрь"
Установка бюджета: еда 15000
Перевод средств: user2 5000
Экспорт данных: finances.csv
